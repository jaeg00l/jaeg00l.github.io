## 문제

선수과목이 존재하는 후위 과목은 다음학기에 들어야한다. A→B→C 이면 A는 1학기, B는 2학기, C는 3학기에 들어야한다. 출력은 1 2 3 이다. 모든 과목의 들을 수 있는 학기를 출력하는 것.

## 접근

위상 정렬을 구한다.

## 답안

```cpp
#include <iostream>
#include <queue>
#include <functional>
#include <vector>

using namespace std;

vector<int> a[1001];
int ind[1001];
int rank_[1001];
int N, M;

int main(void) {
   cin >> N >> M;
   for(int i=0; i<M; ++i) {
      int x, y;
      cin >> x >> y;
      a[x].push_back(y);
      ind[y]++;   
   }

   priority_queue<int, vector<int>, greater<int>> q;
   for(int i=1; i<=N; ++i) {
      if(ind[i] == 0) { 
         q.push(i);
         rank_[i] = 1;
      }
   }

   while(!q.empty()) {
      int x = q.top();
      q.pop();
      
      for(int i=0; i<a[x].size(); ++i) {
         int y = a[x][i];
         ind[y]--; 
         rank_[y] = max(rank_[y], rank_[x] + 1); // y의 순위는 x의 순위 + 1과 기존 y의 순위 중 큰 것
         if(ind[y] == 0) q.push(y);   
      }
   }

   for(int i=1; i<=N; ++i)
      cout << rank_[i] << " ";
   cout << "\n";
   

   return 0;
}
```

### 추가 설명

선수 과목의 학기 수 + 1가 후위 과목의 학기 수가 아닐 수 있다.

max 함수로 `현재 학기 수`와 선수 `과목 학기 수 + 1`를 비교해 더 높은 것을 선택한다.
